# `hello_world.rs`

Compile with:

    rustc hello_world.rs -o hello_world

Run with:

    ./hello_world


# `hello_cargo`

This project was created using `cargo`:

    cargo new hello_cargo --bin

`--bin` makes the project produce executable as opposed to a library
(`--lib`).

Run with (inside the project):

    cargo run

If it was a library:

    cargo build
